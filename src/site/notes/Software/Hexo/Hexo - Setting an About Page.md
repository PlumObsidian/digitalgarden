Setting up "About":

Will be rendered from the "source" directory like so:

```
source/about/index.md
```

index.md will then contain something like so:

```
---
title: About Page
layout: about
index_img: /img/example.jpg
date: 2019-10-10 10:00:00
---
Your blah here..

```
